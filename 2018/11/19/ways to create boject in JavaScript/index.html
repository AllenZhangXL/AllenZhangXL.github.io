Posted on 2018-11-19 | In JavaScript

There are different ways to create Object in JavaScript , there are my note about Creating Object in JavaScript.

  1. Create a new Object:
    var person = {};
    Or

    Create new Object members and functions:
    person['eyes'] = 'hazel';
    person.farewell = function(){alert("Bye everybody!)};

  1. Use Constructor to create Object:

    Use this to create Instance:
    var person1 = new Person('Bob);

  1. Use Object() function to Create new Object:
  1. Use Create() function to create new Instance base on instance(we already have):
    var person2 = Object.create(person1);

These ways, I perfer to use Constructor to create new Object , and I think this way is better to use prototype .
